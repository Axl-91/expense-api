# Expense API


## Description
API for income/expense storing.

The data will be stored in an postgreSQL database

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# developmentin
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
